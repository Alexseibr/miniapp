1. Исправить хедер (верхняя часть)

Сейчас под логотипом слишком много пустого белого пространства.
Сделай:

высоту хедера минимальной (примерно 30% от текущей),

логотип — крупнее, аккуратный, ровно по центру,

убрать текст «Привет, Алексей!» — вместо него должна быть поисковая строка.

Поисковая строка:

стиль iOS,

закруглённая 16px,

placeholder: «Что вы ищете?»,

по нажатию сразу перекидывает во вкладку Лента с применённым поисковым запросом.

2. Обновить иконки категорий

Проблема: «Фермерский рынок» сейчас без иконки.
Нужно:

отрисовать красивую 3D-иконку в стиле остальных (формат PNG/WebP, 1:1),

заменить текущие квадратные иконки на объёмные, плавные, как раньше,

добавить мягкие тени + плавные градиенты.

Иконки должны иметь одинаковый стиль.

3. Перейти на 3-колоночную сетку категорий

Схема:

3 карточки в ряд (3 columns),

адаптив: 2 карточки, если экран узкий,

размер карточки — 160–180px по ширине (по ситуации),

скругления 18–22px,

тень в стиле iOS (soft shadow),

внутри — сверху небольшая 3D-иконка (больше, чем сейчас),

ниже текст категории.

4. Добавить красивый badge количества подкатегорий

Как на iPhone:

справа сверху маленький кружок,

внутри — число подкатегорий,

фон голубой (#2A82F0), текст белый,

размер 18–24px,

обязательно поверх иконки категории.

Пример:
Фермерский рынок
badge: “13”

5. Вернуть стильный визуал категорий (как был раньше)

Сейчас карточки стали квадратные, некрасивые.
Нужно:

вернуть прежний стиль: объёмные, плавные, «дорогие»,

убрать усекающие углы,

всё выровнять по сетке,

добавить одинаковые отступы по 12–18px.

6. Восстановить корректное отображение карточки товара

Когда открываю товар — дизайн сбит. Нужно исправить:

✔ Кнопка «Назад»

сделать её воздушной, iOS-style,

белый круглый фон с тенью,

иконка чёрная/тёмно-серая.

✔ Фото товара

вернуть нормальный рендер фото,

убрать черные/синие поля по бокам,

сделать высоту фото адаптивной,

свайп должен быть без подлагиваний.

✔ Блок продавца

аватар круглый,

имя крупнее,

Используем дизайн-систему iOS: мягкие тени, скругления, лесенка уровней.

Цветовая палитра: голубой #2A82F0 (основной), белый #ffffff, тёмный #1C1C1E.

Шрифт: Inter / SF Pro Display аналог.

Минимум шумов, максимум читаемости.
